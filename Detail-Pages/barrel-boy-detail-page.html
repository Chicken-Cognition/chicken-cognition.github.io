Barrel Boy


Role on Project: Solo Developer

Software Used: Unreal Engine, Blender

Description:

Barrel Boy was my "Dream Game" idea I had before I had started learning how to make games. These days I realise it was overscoped, but I still did some work on it that I'm very proud of, and want to show off.

A lot of these pieces became less about making a game, and more about learning how to make them at all as I went on. Barrel Boy, for example, is how I learned rigging, and my first attempt at making hands in 3d. The hand tree was also how I learned sculpting, and retopology.
